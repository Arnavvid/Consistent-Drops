**Works on Bukkit, Spigot, Paper, Purpur.**

**Recommended versions: 1.18 - 1.21.x.**


## Overview
Consistent drops is an extremely light weight plugin that makes items no longer shoot off in random directions when blocks break. Instead, they fall straight down where the block was mined or destroyed.

Works with normal block drops, chain-breaking blocks like scaffolding and bamboo, and multi-block collapses.

Does not affect player-thrown items or manual drops.
